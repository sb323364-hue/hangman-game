# Игра "Виселица" (Hangman)

## Описание проекта
Простая консольная игра "Виселица", где игрок угадывает слово по буквам на русском языке.

## Инструкция по запуску

### На Linux:
cd ~/'Рабочий стол'/hangman_project
python hangman.py
